# 微博地理位置影响程度可视化工具

一个用于分析和可视化微博地理位置影响程度的工具，基于蜂窝状六边形网格和莫兰迪色系。

## 功能特点

- 基于蜂窝状六边形网格分析微博地理位置数据
- 使用莫兰迪色系可视化影响程度
- 支持时间滑块功能，可按天分析数据变化
- 使用高德地图作为底图
- 提供区域查询功能，可查询特定位置的影响程度
